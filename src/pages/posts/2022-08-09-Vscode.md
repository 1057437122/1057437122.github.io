### how to change nodejs debug port (typescript)

use this command to start node server:`nodemon -x node --loader ts-node/esm --inspect=9230  src/main.ts`
and set debug configuration like this:
```
{
            "name": "Attach",
            "port": 9230,
            "request": "attach",
            "skipFiles": [
                "<node_internals>/**"
            ],
            "type": "node"
        },
```
